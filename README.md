# Arizon RP Launcher 🎮

**تطبيق لانشر احترافي لسيرفر SA-MP RolePlay على أنظمة Android**

## المميزات
- ✅ واجهة عربية احترافية
- ✅ توجه أفقي (Landscape Mode)
- ✅ تصميم عصري وسلس
- ✅ نظام تسجيل دخول آمن
- ✅ إدارة البيانات الشخصية
- ✅ إعدادات الصوت والتحكم
- ✅ أداء عالي وسريع

## المتطلبات
- Android Studio (Arctic Fox أو أحدث)
- Kotlin 1.7+
- Gradle 7.0+
- Min SDK: 24
- Target SDK: 34

## التثبيت
```bash
git clone https://github.com/boursalirayane1-boop/DiaaStudio.git
cd DiaaStudio
# فتح المشروع في Android Studio
```

## الشاشات الرئيسية
1. **Splash Screen** - شاشة جاري التحميل
2. **Home Screen** - الصفحة الرئيسية مع معلومات السيرفر
3. **Profile Screen** - معلومات الحساب والشخصية
4. **Settings Screen** - إعدادات الصوت والتحكم
