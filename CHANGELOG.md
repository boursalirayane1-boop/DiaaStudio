# Arizon RP Launcher - Changelog

## [1.0.0] - 2026-07-22
### Added
- Initial release
- Splash screen with loading animation
- Home screen (Dashboard) with server status
- Profile screen with character information
- Settings screen with audio and control settings
- Bottom navigation between screens
- Arabic language support
- Landscape orientation support
- Responsive design for all screen sizes

### Features
- Real-time server status display
- Player statistics and progression
- Sound volume controls
- Control settings (HUD, Radar toggles)
- Language selection (Arabic/English)

### Technical
- Built with Kotlin
- Target SDK: 34
- Min SDK: 24
- Material Design 3 colors
- RTL support for Arabic
- Custom drawable resources
